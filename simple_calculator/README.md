# Simple Calculator

A basic calculator module that supports addition, subtraction, multiplication, and division.

## Features
- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers (with error handling for division by zero)

## Prerequisites
- Python 3.8 or later
- Poetry
